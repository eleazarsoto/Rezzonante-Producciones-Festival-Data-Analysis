# Rezzonante Producciones — Festival Data Analysis

## Context

Rezzonante Producciones is a production company focused on jazz and improvised music, which I founded in 2018 in Ajijic, Mexico. Through projects like the El Lago Suena international festival, I turn years of real event data into programming, venue, and budget decisions.

## Business Questions

- Which venues and event formats generate the highest attendance?
- How does attendance vary by month and season?
- Which programming decisions correlate with audience growth?

## Data

Historical records from festival editions (2018–2025): events, venues, dates, artists, formats, and attendance.

Data is captured in a structured workbook (`Rezzonante_BD_Conciertos`) and migrated to a relational database for analysis.

> **Status:** Data entry in progress. Migration to Google Sheets and SQL analysis phase coming next.

## Methodology

1. **Data cleaning and structuring** — Google Sheets → SQL
2. **Exploratory analysis** — SQL queries and Python (Pandas)
3. **AI-assisted workflow** — I use LLMs to accelerate query writing and validate logic; business questions, interpretation, and final decisions are my own
4. **Visualization and reporting**
